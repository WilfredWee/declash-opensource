About DeClash

Bringing Debaters Together

What is DeClash?

The mission of the DeClash project is to bring debaters together. In its current form, DeClash provides an easy-to-use service to make debating practices faster and more efficient. Training directors can set up a practice session, debaters can register online, and teams, rooms and speaking positions can be automatically allocated through an algorithm. All in just a few clicks.

DeClash is a continuation of a project of mine which was an Excel VBA Macro with the use of Google Forms for online registration. The project was used by the UBC Debate Society in late 2013 with success. So I decided to extend its capabilities by bringing the service to the cloud, enabling more opportunity for growth.

Currently, DeClash supports 3 formats: British Parliamentary, Asian Parliamentary(or any 3v3), Canadian Parliamentary(or any 2v2 with teams being able to select their position preferences.)

If you are interested in using this for your club, please contact enquiry@declash.com.


Still In Development

As this website is still under heavy-development, do expect some bugs to appear. If anything is not working right, please send an email to feedback@declash.com with a description of what went wrong.

There are a few critical functions that I would like to implement as soon as possible. One of them is room pairing preference.

More importantly, if there are some functions that you would really like in this website, please do email feedback@declash.com and I'll definitely see how I can deliver that!


Strategy & Promise

Currently, the website is free as it is still under a testing phase, and Amazon provides a free year of hosting. Going forward, if there is enough demand for this service, I will set up some kind of a monetization strategy to support server hosting. This will most likely be in the form of a monthly subscription but nothing is decided yet.

My promise to you: If I do decide to discontinue development of this service, I will open source all the code for this website and hopefully somebody else is able to support this service after me.


Forward

My main focus now will be to improve the reliability of my algorithm, add important features from your suggestions and improve the User Experience for the website.

Moving forward, I am seriously considering providing a new service for tournament tabbing. I've tabbed a few tournaments before and have found that many of the tabbing softwares fall short in terms of usability. I might want to improve that situation, depending on how much time I am able to devote to this project, as well as the project's reception.



Finally, enjoy! Looking forward to your feedback. :)
